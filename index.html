Kalkulator Simplex

Masukkan masalah optimasi linier dan klik solve. Cost function untuk memaksimalkan atau meminimalkan harus menggunakan kata "min" atau "max". Batasan harus berupa pertidaksamaan atau persamaan (<=, >=, atau =). Semua variabel harus berada di sisi kiri semua persamaan. Semua persamaan harus berada pada garis yang terpisah. Sebagai contoh:

max 3x1 + x2 + 3x3 2x1 + 3x2 + 5x3 <= 64 x1 - x2 - x3 >= 0 x2 - x3 >= 2

atau

min 12x1 + 16x2 x1 + 2x2 >= 40 x1 + x2 = 30 * >= 0

Anda juga dapat menggunakan operator * agar batasan diterapkan ke semua variabel. Sebagai contoh:

min 5x1 - 20x2 -2x1 + 10x2 <= 5 2x1 + 5x2 <= 10 x1 + * >= 0

akan menjadi

min 5x1 - 20x2 -2x1 + 10x2 <= 5 2x1 + 5x2 <= 10 x1 + x1 >= 0 x1 + x2 >= 0

Yang ditampilkan di bawah ini adalah setiap tableau yang digunakan untuk menghitung hasilnya.
